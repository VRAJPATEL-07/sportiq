# Privacy Policy

**Effective Date:** January 2026  
**Application Name:** SportiQ – Smart Sports Equipment Management & Borrowing Mobile Application

SportiQ (“we”, “our”, “us”) is committed to protecting the privacy of its users. This Privacy Policy explains how we collect, use, store, and protect user information when using the SportiQ mobile application.

## 1. Information We Collect

We may collect the following information from users:

### 1.1 Personal Information
- Name
- Email address through Google Sign-In
- User role, such as Student or Admin

### 1.2 Usage Information
- Equipment borrowing records
- Return status
- Borrowing and return timestamps
- Penalty-related information

### 1.3 Device and App Information
- Device type
- Basic app usage logs for debugging and improvement

## 2. How We Use Information

The collected information is used to:

- Provide access to the SportiQ application
- Manage equipment borrowing and return activities
- Track equipment availability
- Maintain user accountability
- Improve system performance and user experience
- Prevent misuse or unauthorized access

## 3. Data Storage and Security

User data is stored securely in a MongoDB cloud database. Authentication is handled through Firebase Authentication. Access to data is restricted based on user roles, such as Student and Admin.

We take reasonable technical and organizational measures to protect user information from unauthorized access, misuse, or loss.

## 4. Data Sharing

SportiQ does not sell, rent, or share personal user data with third parties. User data is used only for application functionality and institutional equipment management purposes.

## 5. User Rights

Users may request to:

- View their stored information
- Correct incorrect information
- Request deletion of their account or records, subject to institutional rules and admin approval

## 6. Data Retention

User and borrowing records may be retained for administrative, reporting, and accountability purposes as required by the institution.

## 7. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any major changes will be communicated through the application or institution notice.
